The Big List of Communism
A web directory of the Communist Left
Help build a list of works by the different currents of the Communist Left! The Big List of Communism (TBLOC) is a project to build a web directory containing all the communist websites on the net. Big or small, we hope you'll find them here.

TBLOC relies on community input to map out relevant websites on the net. To help, submit a form below.
